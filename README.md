# 数据分析与可视化项目总入口

这里是 TreasureLZ 的数据分析与可视化项目总入口，面向本科、专科学生和 Python / Django 初学者，整理适合课程设计、毕业设计、项目练习和二次开发的开源项目。

本仓库只做“总入口、项目索引、学习路线、维护规范和选题规划”，不在首页堆放项目截图。每个项目的详细说明、运行步骤、截图展示和常见问题，都放在对应项目仓库中维护。

## 入口定位

这个仓库解决三个问题：

- 让访问者快速知道有哪些项目可以参考
- 帮学生按难度、方向和用途选择项目
- 给后续新增项目提供统一的文档和维护规范

如果你是第一次访问，建议先看“项目目录”和“如何选择项目”；如果你准备贡献或继续扩展项目，建议看 `docs/` 目录中的仓库建设建议和 README 模板。

## 状态说明

| 状态 | 含义 |
| --- | --- |
| 活跃开发 | 正在新增功能、补充数据或调整架构 |
| LTS 维护 | 主要保证项目可读、可运行、可学习，修复文档和明显运行问题，不做大规模重构 |
| 规划中 | 选题已确定，但代码和文档尚未完成 |
| 暂停维护 | 暂不更新，仅保留历史参考 |

当前三个老项目都建议按 LTS 方式维护：保留已有 star 和搜索价值，继续补文档、补运行说明、修明显问题，把主要精力放到新入口和后续轻量数据分析项目。

## 如何选择项目

| 目标 | 推荐项目 | 原因 |
| --- | --- | --- |
| 第一次接触 Django 数据分析项目 | 图书分析大屏展示系统 | 业务容易理解，功能完整，适合从页面、模型、数据处理开始学习 |
| 想做行业价格或商品行情分析 | 药材数据可视化系统 | 数据主题明确，图表集中，容易替换成其他行业数据 |
| 想做毕业设计或综合看板 | 农业生产可视化系统 | 同时包含农业数据和气象数据，扩展空间更大 |
| 想练习大屏和可视化展示 | 图书分析大屏展示系统、农业生产可视化系统 | 包含多类统计图、地图和大屏展示页面 |

## 推荐学习路线

1. 先从本仓库选择一个和自己课题接近的项目。
2. 进入对应项目仓库，阅读 README、LTS 说明和常见问题。
3. 按项目 README 配置 Python、MySQL、依赖和数据库。
4. 先跑通原项目，再阅读数据采集、数据清洗、模型、视图和模板。
5. 替换自己的数据集，完成一次“同结构换主题”的改造。
6. 增加新的统计指标、筛选条件、图表页面或导出功能。
7. 最后整理项目说明、运行截图、演示视频和答辩材料。

## 后续建设方向

后续这个入口会继续扩展项目数量，但每个新增项目都应优先满足这些要求：

- 能说明清楚项目目标、数据来源和适用场景
- 能提供稳定的本地运行步骤
- 能区分“快速运行”和“重新采集/清洗数据”两条路径
- 能提供截图、常见问题和二次开发建议
- 能说明维护状态，避免访问者误解项目是否还在大改

建议新增项目逐步从重 Django 系统扩展到更轻量的分析路线：

- `pandas + Jupyter Notebook`：适合教学和分析过程展示
- `pandas + Streamlit`：适合快速做交互式数据看板
- `Django + MySQL + ECharts`：适合毕业设计和完整 Web 系统
- `公开数据 + 可视化报告`：适合课程作业、答辩和作品集
